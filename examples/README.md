# Example Files

This folder contains **illustrative, AIA-style reference examples** intended to demonstrate how pay application data is commonly structured and calculated.

These files are **not licensed AIA forms** and are provided for educational purposes only.

---

## g703-continuation-sheet-example.csv

A realistic, internally consistent example of a G703-style continuation sheet.

Key characteristics:
- Line items based on a typical commercial project
- Includes prior work, current period work, and stored materials
- Uses a **10% retainage rate** (common, but contract-dependent)
- Demonstrates partial, full, and zero-completion line items
- All math is internally consistent for validation and testing

Intended uses:
- Reference for understanding G703 structure
- Import testing for billing software
- Math validation and reconciliation examples

---

## sample-sov.csv

A simplified Schedule of Values derived from the same line items used in the G703 example.

Intended uses:
- SOV setup reference
- Import examples
- Understanding how SOV values roll into pay applications

---

## JSON Examples

Where provided, JSON files mirror the CSV examples and are intended for:

- Testing calculations programmatically
